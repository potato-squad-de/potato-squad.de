# [potato-squad.org](https://potato-squad.org)

## Usage

```
NAME:
   site - potato-squad.org Site

USAGE:
   site [global options] command [command options] [arguments...]

COMMANDS:
   run, start, serve  Run the site
   help, h            Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --discord-invite value, --discord-link value  Discord invite URL [$DISCORD_INVITE_URL]
   --help, -h                                    show help
```
